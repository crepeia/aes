Métodos de parar

Existem dois métodos para parar de fumar: de uma vez ou aos poucos.

Você deve escolher o que achar melhor. Contudo, pesquisas mostram que decidir parar de uma vez funciona melhor do que parar aos poucos.

Parar de uma vez

  • Parar de uma vez significa parar de fumar totalmente.
  • No próximo passo você poderá marcar sua data para parar.
  • Estudos mostram que marcar uma data para parar é fundamental para o sucesso do tratamento.

Parar aos poucos

  • Parar aos poucos significa diminuir o número de cigarros fumados por dia até parar totalmente.
  • Não existe nível seguro para o uso de cigarros.
  • Você deve parar totalmente até 2 semanas após começar a parar aos poucos.
  • Qualquer número de cigarros fumados apresenta riscos a sua saúde.
  • Na próxima etapa, você deverá indicar uma data em que terá parado totalmente.

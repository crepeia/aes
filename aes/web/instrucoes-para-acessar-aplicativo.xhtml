Gostaria de uma ajuda para fazer a instalação do aplicativo, o seu cadastro e entrar no chat?

Caso não precise de ajuda, basta acessar o link

Caso precise de ajuda, siga o passo a passo abaixo:

  1. Entre no link
  2. Instale o aplicativo no seu aparelho
  3. Abra o aplicativo e será direcionado para a página de login
  4. Faça seu cadastro
  5. Abra a lista de abas para acessar a aba do chat
  6. Acesse a aba de chat
  7. Conecte-se ao chat
  8. Envie para o chat a mensagem #{messages['instrucoes.para.acessar.aplicativo.p.1']}

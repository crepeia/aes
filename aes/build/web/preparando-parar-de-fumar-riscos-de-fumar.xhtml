Riscos de fumar

O cigarro causa diversas doenças para o fumante e para as pessoas que respiram a fumaça indiretamente. Veja abaixo algumas das diversas doenças:

Curto prazo

  • Baixa resistência física,
  • Piora da bronquite e asma,
  • Risco de impotência.

Longo prazo

  • Problemas no coração - infarto,
  • Derrame cerebral,
  • Diversos tipos de câncer (boca, laringe, pulmão, bexiga, pâncreas, ...)

Lembre-se!

  • Existem riscos para quaisquer números de cigarros fumados;
  • O uso de cigarros light não traz benefícios para sua saúde.
